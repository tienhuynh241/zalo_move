-----------------Tool hỗ trợ chuyển dữ liệu Zalo sang ổ khác----------------
- Hỗ trợ cài đặt trên windows
- Tool chuyển toàn bộ dữ liệu media ẩn của Zalo bao gồm: File, ảnh, video,...
- Chỉ cần chọn vị trí lưu sau đó ấn bắt đầu là xong
*Lưu ý: Sau khi di chuyển, Zalo sẽ sinh ra thư mục ZaloMedia trong ổ chuyển. Nếu xoá sẽ mất hết dữ liệu Zalo.
- Nên tắt Zalo trước khi thực hiện để tránh lỗi
